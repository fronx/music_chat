# CoderDojo-Projekt: Chat

Starte die Python-Konsole von einem Terminal aus:
```
python
```

Lade die "base"-Bibliothek:
```python
import base
```

Lies die Hilfe für "base":
```python
help(base)
```

## Einfacher Chat

Bau deinen eigenen Empfänger mit Hilfe der `receive`-Funktion.
Bau deinen eigenen Sender mit Hilfe der `send`-Funktion.

## Musik-Chat

Baue einen Empfänger, der Chat-Nachrichten als Musik interpretiert und abspielt.
Mit der `play`-Funktion kannst du Töne produzieren.
